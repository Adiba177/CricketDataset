# 🏏 Cricket World Cup Dataset (2019)

## 📌 Project Overview
This dataset was collected and used as part of my Final Year Project (FYP).  
The main goal of this project was to analyze cricket data and build a machine learning model for performance and score prediction.

## 📊 Dataset Description
This dataset contains cricket match data from the 2019 World Cup.  
It includes detailed match statistics used for analysis and prediction tasks.

## 🎯 Purpose of Dataset
- To perform data analysis on cricket matches
- To train machine learning models (e.g., Random Forest, Linear Regression)
- To predict match outcomes and scores
- To support academic research in sports analytics

## 📁 Columns (Features)
The dataset includes features such as:
-Home(1)/Away(0)
- Ball
- Wickets
- Ranking
- Batting
- Avg Score
- Total Avg score
- Toss
- Player Ranking
- Runs
## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
- GitHub for version control

## 📌 Note
This dataset is part of an academic FYP and is used for educational and research purposes only.
